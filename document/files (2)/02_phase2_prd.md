**6.1 PWA技术规格**

  ---------------------- ------------------------------------------------------------- -------------------------------- ------------
  **PWA特性**            **规格**                                                      **实现方式**                     **优先级**

  **可安装性**           「添加到主屏幕」提示（iOS Safari / Android Chrome）           manifest.json + service worker   P0

  **图标与启动画面**     512×512 PNG图标，品牌色启动画面                               manifest icons配置               P0

  **离线支持（基础）**   已访问的新闻/指南页离线可读（缓存）                           Service Worker Cache API         P0

  **离线支持（高级）**   论坛草稿离线保存，联网后自动提交                              IndexedDB + Background Sync      P1

  **Push通知**           浏览器Push通知（需用户授权）                                  Web Push API + VAPID密钥         P0

  **性能优化**           FCP \<1.5s，LCP \<2.5s，CLS \<0.1（Core Web Vitals达标）      代码分割+图片懒加载+ISR          P0

  **iOS限制说明**        iOS Safari的Push通知需iOS 16.4+；需引导用户升级或用邮件替代   降级方案：邮件通知               P0
  ---------------------- ------------------------------------------------------------- -------------------------------- ------------
