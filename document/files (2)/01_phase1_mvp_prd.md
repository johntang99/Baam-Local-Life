**1.3 技术约定**

  ------------------ ----------------------------------------------------------------------------------------
  **项目**           **约定**

  框架               Next.js 15 App Router，TypeScript，Tailwind CSS

  渲染策略           新闻/指南/商家详情页：SSR（SEO优先）\| 首页：ISR（revalidate 300s）\| AI搜索/论坛：CSR

  数据库             Supabase PostgreSQL + pgvector（向量搜索）

  AI接口             Claude API --- Sonnet处理复杂任务，Haiku处理批量摘要/翻译

  图片               Cloudflare R2存储，Next.js Image组件优化

  认证               Supabase Auth（Email + Google OAuth + 微信OAuth预留）

  语言切换           URL前缀：/zh/\... 和 /en/\... \| 默认根据浏览器语言判断

  响应式             Mobile-first，断点：375 / 768 / 1280px
